Student Management System (Python, OOP, Modular Architecture)

This is a fresh starter Student Management System (SMS) written in Python. It is CLI-based and uses JSON files for persistence.
NumPy, Pandas and SciPy are optional extras used for analysis and CSV import/export if installed.

## Features
- Add / edit / remove students
- Add / edit / remove courses
- Enroll students in courses and assign grades
- View student details and transcripts
- Simple user authentication (admin / student)
- JSON-based persistence with automatic backups
- Reports: average grades, top students, grade distributions
- CSV import/export for students (Pandas if available)
- Unit tests (pytest)
